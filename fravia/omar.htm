This is a list of all the decent free page providers I was able to find with some serching! If you know of any other (free) providers that are not listed here (which you probably do), please e-mail me! I didn't even bother to list the so called "other goodies", since I believe that if you found Fravia+'s site, you know enough about the Web to not need those stupid on page html builders or any of that crap like FrontPage, Composer and so on. And if you don't know, go search the Web about "html guide*" and LEARN!
 
  • Geocities: 3Mb of web space, a fake (remailer) e-mail address and you have to display some advertising for them.
  • Fortunecity: Same as geocities, but with 10Mb web space.
  • Angelfire: Same as above, but with an ugly 200Kb space.
  • Naz: 2Mbs, and advertising.
  • Phrantics Trailepark: 3Mb. They're not taking new accounts right now, but will again soon!
  • Tripod: 2Mb's.
  • Xoom: 5Mb's, e-mail (fake of course) and so on.
  • 123 go global: somewhat complicated, but check it out anyway. It's fast.
  • BIP: not in english, so I wasn't really able to find my way around.
  • Hong Kong Cybercity: these guys just opened, and seem very good. 5Mb's
  • Europe@home: 2mb's. Various languages.
  • Ezpage: doesn't seem to have a limit (not sure though). Seems good.
  • Expage: also doesn't seem to have any limit. Check it out.
  • Freetown.
  • Student Advantage Network: good, but you need something called: A Student Advantage ID. Anyway, I think you can get that id on the site itself.
  • Naples.
  • Market Net Free Web Pages.
  • OneSky: 300Kb, unlimited hits.
  • Page List: seems unlimited space.
  • Phrantic Project.
  • Town Square 2000: create up to 3 pages here.
  • Web Spawner: also seems to have no limit.
  • Chez: 10Mb.
  • China Info: 100Kb.Commercial pages accepted for free.
  • Focus-Asia: 5Mb.
  • Free-org: 3Mb.
  • Hypermart: fast, 10Mb.
  • Indocities: 2Mb.
  • Pro Hosting: wow! 50Mb, full CGI (means you can make your own) and e-mail. Also, seems reasonably fast.
  • Rampage: 100Mb, full cgi, e-mail and DOMAIN: www.yourname.com. All free, Downfall: somewhat slow.
  • Tao Tree: 3Mb.
  • Yclub: 2mb.
  • CrossWinds: Unlimited space, REAL e-mail, but slow as hell!
  • Dark Castle: also unlimited space, but you have to send a written snail-mail registration, and a US$ 2.00 fee (I consider this being for free).
  • Earth On Line: 6Mb. Temporarily down, but will be back on-line soon.
  • IWantFreeStuff: 3Mb, also comercial.
  • Free Yellow.
  • ForFree: 2Mb.
  • Net Taxi: 1Mb.
  • Internet Club: 6Mb, probably also comercial.
  • Kostenlose Private Homepages: 1Mb.
  • Vega Net: 10Mb.
